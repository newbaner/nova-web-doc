介绍 ​

此模板体系为使用 Vue 3 构建现代 Web 应用程序提供了坚实的基础，包括高效的路由、状态管理、UI 组件、数据可视化、HTTP 请求、CSS 样式选项和常见编程任务的实用库。通过精心策划的工具和库的精选，增强您的开发体验，加快项目的推进。

技术栈 ​

了解并熟悉框架使用到的技术栈，能让你使用本框架更得心应手。

  • Vue3 流行的 JavaScript 框架的最新版本，用于构建交互式 Web 界面。
  • Vue Router Vue.js 的官方路由器，为单页面应用程序提供无缝导航。
  • pinia 提供强大的状态管理解决方案的 Vue 3 存储系统
  • Vite 用于现代 Web 开发的轻量级快速构建工具，使您的开发流程更加高效
  • Element Plus Vue 3 UI 库，提供各种组件，用于创建时尚且响应式的用户界面
  • axios 基于 Promise 的 HTTP 客户端，用于从您的 Vue 应用程序发出 API 请求
  • TailwindCSS (2选1) 根据项目需求选择 TailwindCSS 用于实用型 CSS 或 UnoCSS 用于轻量级替代。
  • UnoCSS (2选1) 高性能且极具灵活性的即时原子化 CSS 引擎
  • vite-plugin-pages Vite 插件，可从文件系统结构自动生成路由，简化路由设置
  • Echarts 强大的图表和可视化库，用于创建交互式数据可视化
  • lodash 便捷的实用程序库，提供各种函数，简化 JavaScript 编程
  • currency.js 处理货币格式和计算的库，解决计算精度问题
  • dayjs 轻量级快速日期处理库，用于格式化、解析和显示日期
